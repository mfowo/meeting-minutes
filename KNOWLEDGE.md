# ノウハウ・フィードバック蓄積

このファイルは議事録生成の品質向上のために、フィードバックやノウハウを蓄積するファイルです。
新しい学びは日付付きで追記してください。

---

## 表記ルール

### 数字・漢字の使い分け
- 「六大学」→ 漢字表記が正しい（「6大学」は誤り）
- 「六大」→ 六大学の略称（「6台」は誤り）

### 馬術用語
- **失権**：競技の途中でコースを外れた場合の公式用語。「途中棄権」は誤用。
- **先通し**：コースを先に走ること。「先行出走」は誤り。

### 地名
- **つま恋**：静岡県の施設名。「妻恋」は誤変換。

## ネクストアクションの抽出精度

### 見落としやすいパターン
| 表現パターン | 例 |
|---|---|
| 「〜しておきます」「〜しますね」 | 「明日朝一にメールしておきます」 |
| 「〜してもらっていいですか」→「はい」 | 質問→承諾の対話形式 |
| 「〜ということでよろしいですか」→「はい」 | 確認→合意の流れ |
| 「〜お願いできますか」→「わかりました」 | 依頼→受諾 |
| 誰かが名指しで言われて「あ、はい」と答える | 「高島さん、これ更新しておいて」→「わかりました」 |

**日程・期限の言及**
- 「○日までに」「翌日」「今週中に」「速やかに」
- 「○時に」「朝一に」「終わり次第」

**副次的に決まるタスク**
メインの議題を決めた際に付随して生まれる小さなタスクを見落としやすい。
例：「フレンドシップ廃止」→ 副次タスク「伊藤くんに連絡・了承を得る」

## 議事録フォーマット

### ネクストアクション表のフォーマット
```markdown
| 担当者 | 期限 | タスク |
|--------|------|--------|
| 渡邊   | 翌日 | 〇〇への連絡 |
```

「速やかに」「大学役員確定後」など期限が曖昧な場合も明記する。

### 「持ち越し課題」セクションを設ける
決定事項・アクションアイテムに含まれない、次回以降で議論・確認が必要な事項を別セクションに記録する。

## 2026-03-15 運営ミーティング フィードバック

### 馬名の誤変換パターン（今回判明）
| 正しい表記 | 誤変換例 | パターン |
|---|---|---|
| ラヴィ | ラビー | 長音「ヴィ」→「ビー」への変換 |
| ビオンデッツァ | ビヨンデッツァ | 「ビオ」→「ビヨ」への変換 |
| デレクトゥール | ディレクトゥール | 「デ」→「ディ」への変換 |
| ブエナベントゥーラ | ブーナベントゥーラ | 「ブエナ」→「ブーナ」への変換 |
| リバプ | リバップ | 促音の誤挿入 |
| タオ | TAO | アルファベット表記のまま残る |

**パターンまとめ**：外来語の馬名は「母音の伸ばし方」「半濁音・濁音の区別」が特に誤変換されやすい。

### 人名の誤変換パターン（今回判明）
| 正しい表記 | 誤変換例 | 理由 |
|---|---|---|
| 高橋駿人（しゅんと） | 高橋春人、俊人 | 同音の当て字 |
| 渡邊 | 渡辺 | 旧字体の認識不足 |
| 中嶋先生 | 中島先生 | 旧字体の認識不足 |
| 宮澤 | 宮沢 | 旧字体の認識不足 |
| 須江 | 末 | 略称をそのまま使用 |
| 木場 | 小葉、コバ | 呼称・当て字 |

**パターンまとめ**：旧字体（邊・嶋・澤）は要注意。呼称・略称が正式名称として記載されないよう、glossary.csv の「別名」欄に登録しておく。

### 馬術用語
- **失権**：「途中棄権」は誤用。競技の公式用語は「失権」。
- **先通し**：「先行出走」は誤り。コースを先に走ることを「先通し」という。

### 表記ルール
- 「六大学」「六大」は漢字表記。数字（6大学、6台）は誤り。
- 施設名「つま恋」は平仮名。「妻恋」は誤変換。
